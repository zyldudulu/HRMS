# HRMS
企业人力资源管理系统
### 功能需求

1. 用户管理
2. 员工信息管理
3. 工资管理
4. 考勤管理
5. 公告管理

- 管理员
  - 用户管理
    - 登录系统
    - 修改密码
    - 退出登录
    - 重置密码
  - 员工信息管理
    - 添加员工信息（Excel批量导入）
    - 删除员工信息（批量）
    - 修改员工信息
    - 查询员工信息（多条件查询）
    - 批量导出员工信息
  - 考勤管理
    - 查询考勤信息
    - 查询请假信息
    - 审核请假信息（同意、拒绝、撤销操作）
  - 工资管理
    - 工资结算
    - 添加奖惩记录
    - 删除奖惩记录（批量）
    - 修改奖惩记录
    - 查询奖惩记录
  - 公告管理
    - 添加公告
    - 删除公告（批量）
    - 修改公告
    - 删除公告

- 员工
  - 用户管理
    - 登录系统
    - 修改密码
    - 退出登录
    - 上传头像
  - 员工信息管理
    - 查看个人信息
  - 考勤管理
    - 签到、签退
    - 请假
  - 工资管理
    - 查看工资（已结算、未结算）
    - 查看个人奖惩记录
  - 公告管理
    - 查询公告
